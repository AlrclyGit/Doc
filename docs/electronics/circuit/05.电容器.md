# 电容器

电容（C）:由两个金属极片中间隔绝缘介质构成的能够存储电荷的元器件

- 无极性电容：会标注「容量、电压、温度」，同值代换，电压可以略高，耐高的容易坏。
- 有极性电容：铝（+） -- 电解液 -- 氧化铝（-），在无极性电容的基础上会标注正反，比无极性电容容易坏。

## 单位：法拉 F

$1\text{F} = 10^{6}\mu\text{F} = 10^{9}\text{nF} = 10^{12}\text{pF}$

## 决定电容量单位的因素

1. 间隔距离：距离越小容量越大。
2. 接触面积，面积越小容量越小。
3. 材料：介质和金属材料

## 单位的标称方法

1. 直接标称：电解电容。$330\mu\text{F}$
2. 字母数字混合标称。无极性。$3P3 = 3.3\text{pF}$ $6n8 = 6.8\text{nF}$
3. 倍率标称（单位为 pF）：
   - 两位 $22 = 200\text{pF}$
   - 三位 $333 = 33000\text{pF} = 33\text{nF}$（末位是 0 的个数，单位是 PF）
4. 带小数点的默认单位是 $\mu\text{F}$

## 电容的特性

- 瞬间通电流（漏电）
- 隔直流、通交流（极性电容）
- 通高频，阻低频（无极性电容）
- 电容器是交流电的纯电阻
- 充放电（存储电荷）

### 公式

- 电容对交流电的阻碍作用公式：$X_C = \frac{1}{2\pi fC}$
  - $X_C$：容抗（单位：欧姆 $\Omega$）
  - $\pi$：圆周率（约 3.1416）
  - $f$：交流电的频率（单位：赫兹 Hz）
  - $C$：电容的容量（单位：法拉 F）
- 频率越高、电容量越大，电容对交流电的阻碍作用越弱
- 容抗与交流电频率 $f$、电容量 $C$ 成反比
- 电容对直流电，$f = 0$，故 $X_C \to \infty$，电容相当于开路。

## 电容的应用

1. 交流降压：根据容抗。
2. 并联分流保护：开机瞬间保护（高频耦合容器）。元器件上并联的无极性电容器
3. 高频耦合：通高阻低（无极性）：信号通道中串联的无极性（小容量）电容器
4. 低频耦合：隔直通低（电解电容）：信号通道中串联的电解电容
5. 高频旁路：对地通高频，保留低频（无极性）：信号线对地并联无极性电容器
6. 直流滤波：滤除纹波。（电解电容）：对地并联的有极性电解电容
7. 火花吸收：无极性。（静电吸收）：直流电对地并联无极性电容器
8. 振荡器定时：利用电容器的充放电特性。振荡器对地或供电并联无极性电容器

## 电容的故障

1. 无容量：并联同规格电容器判断好坏
2. 容量减小：并联同规格电容器判断好坏
3. 短路：电阻值为零
4. 漏电大：代换同规格电容器判断好坏
