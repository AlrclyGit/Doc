# 电感器

电感器（L）：由漆包线或纱包线绕制而成的线圈、又叫高频扼流圈或阻流圈

## 电感单位

$\text{H（亨利）}$、$\text{mH（毫亨）}$、$\mu\text{H（微亨）}$、$\text{nH（纳亨）}$

$1\text{ H} = 1000\text{ mH} = 10^6\mu\text{H} = 10^9\text{ nH}$

## 决定电感量的因素

匝数、线径、松紧、材料，材料的多少

## 电感的特性

### 楞次定律

瞬间电流流过电感器时，电感器会对电流产生阻碍作用----感应电动势（其方向与电流方向相反），同时生成感应电压。

### 法拉第定律

1. 电感线圈做切割磁力线运动时会产生交流电（发电机）
2. 电感器流过电流时在其周围会形成磁场：
   - 交流电产生交变磁场
   - 直流电产生固定磁场

## 电感器的应用

### 直流电中

对直流电流几乎无阻碍（仅线圈直流电阻）

电感器可以恒流、滤波，使用的时候串联于直流电中，*恒流滤波电感*。

### 交流电中

**感抗：阻高频通低频**，电感器是交流电的纯电阻，下面是感抗的基础定义式（角频率形式）

$X_L = 2\pi f L = \omega L$

- $X_L$ = 感抗（$\Omega$）
- $L$ = 电感（$\text{H}$）
- $f$ = 频率（$\text{Hz}$）
- $\omega$ = 角频率（$\text{rad/s}$）
- $2\pi f = \omega$

### 动能转电能

发电机

### 电能转动能

电动机、电磁铁、电磁阀、电磁炉、接触器（强电控制强电）、继电器（弱电控制强电）

![继电器](https://cos.alrcly.com/2025-12-07-zeVhPl.webp)

### 逆变器（DC 变 AC）

- 逆变器的工作条件：
  - 直流电
  - 储能电感器
  - 开关器
  
- 逆变器的优点：
  - 直流升压（并联）
  - 逆变为高频交流再整流降低纹波系数
  - 多路输出
  
### 交流降压

交流电中使用感抗降压

## 常见故障

- 短路
- 匝间短路。所以万用表测得不短路，不一定就代表电感是好的。
