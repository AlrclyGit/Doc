# 认识音程与音程计算的解题技巧

## 音程 Interval

*定义*：**音程**是指两个音之间的距离。

英文术语：What's the degree between C and D.（C 到 D 的音程是多少？）

一般提问「C 到 D 的音程是多少？」的时候，其实是在问，D 在 C 为主音的音阶的的第几级。
所以，相比传统的「大 2 度」翻译，其实称为「第二级」更合理。

英文里 $C_{4}$  到 $C_{4}$ ，它不叫 first，它叫 unison。

英文里 $C_{4}$  到 $C_{5}$ ，它叫 Octave。也就是我们中文里的「一个八度」。

![2026-03-18-CeVFEs](https://cos.alrcly.com/2026-03-18-CeVFEs.jpg)

大小调不一样的音（蓝色），在大调里的为「大 N 度」，小调里的为「小 N 度」，相同的（红色）为「纯 N 度」。另外，二度的音为特例，大小调都叫「大二度」。C 到降 D 才叫做「小 2 度」。

- 2、3、6、7 分大小
- 1、4、5、8 叫作纯
- 增 4 度是 C 到 $F_{♯}$，减 5 度是 C 到 $G_{♭}$
- 增 3 度是 C 到 $E_{♯}$, 减 3 度是 C 到 $E_{♭♭}$

## 音程计算

以 D 到降 A 举例：

- *先算级数*：D ~ A = 5 级
- *口诀*：**纯 小大小大 纯 (增减) 纯 小大小大**
- 口诀：P1 m2M2m3M3 P4「A4D5」P5 m6M6m7M7
- *判定性质*：D ~ A 数 7 下到「A4D5」，而级数为 5，所最后结论为减 5 度
