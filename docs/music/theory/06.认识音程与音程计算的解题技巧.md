# 认识音程与音程计算的解题技巧

## 音程 Interval

相比传统的「大 2 度」称为「第二级」更合理。

C 到 D 的音程是多少？———— 就是在问，D 在 C 为主音的音阶的的第几级。

一个八度：英文称为 Octave。

![2026-03-18-CeVFEs](https://cos.alrcly.com/2026-03-18-CeVFEs.jpg)

大小调不一样的音（蓝色），在大调里的为「大 N 度」，小调里的为「小 N 度」，相同的（红色）为「纯 N 度」。另外，二度的音为特例，叫大二度。

- 2、3、6、7 分大小
- 1、4、5、8 叫作纯
- 增 4、减 5 为半音
- C 到重降 E 为减三度
- C 到升 E 为增三度

## 音程计算

以 D 到降 A 举例

### 先算级数

D ~ A = 5 级

### 判定性质

- 纯 小大小大 纯 (增减) 纯 小大小大
- P1 m2M2m3M3 P4「A4D5」P5 m6M6m7M7

D ~ A 数 7 下到「A4D5」，而级数为 5，所最后结论为减 5 度
