# 第五课 大小调音阶及调号

## 音阶

- 大调音阶：全全半全全全半。
- 关系（自然）小调音阶：大调的第六个音开始，全半全全半全全。
- 和声小调：关系小调升第 7 级音叫和声小调。
- 旋律小调：关系小调仅仅在上行时升第 6、7 级音叫旋律小调。
- 爵士乐：爵士乐框架内旋律小调是关系小调升第 6、7 级音，下行也升。

## 调号

大调

- 7(白键) + 5（黑键） = 12
- 14（升降号调） + 1（C 大调） - 3（重复） = 12
- B♭ = F♯
- B = C♭
- C♯ = D♭

小调

- 默认按照 关系（自然）小调 标注调号
- 其他小调临时标记

## 顺阶和弦表

- 在音阶的书写里，不出现重复的音名。
- 数字 (1-7)：代表简谱中的音阶位置（级数）。
- “大”与“小”：指的是在这个音位上构建的和弦属性。
  - 大 (Major)：大三和弦（听起来亮丽、开阔）。对应 1、4、5 级。
  - 小 (Minor)：小三和弦（听起来忧郁、柔和）。对应 2、3、6 级。
  - Dim (Diminished)：减和弦（听起来不和谐、有张力）。对应 7 级。

### 查音阶

如果你想知道“A 大调”有哪些音？

找到 A 开头的那一行，横着读：A, B, C♯, D, E, F♯, G♯。这就是 A 大调的 1 到 7 级音。

### 找和弦（最常用的功能）

如果你在写歌或者写伴奏，想知道“D 大调”里自然好听的和弦有哪些？

找到 D 这一行：

- D (1 级，大)
- Em (2 级，小)
- F♯m (3 级，小)
- G (4 级，大)
- A (5 级，大)
- Bm (6 级，小)
- C♯dim (7 级，减)

### 转调（移调）

如果你有一首歌是 C 调的，和弦进行是 C - Am - F - G（即 1-6-4-5 级）。

现在你想改成 G 调唱：

看 C 行：1=C, 6=A, 4=F, 5=G。

看 G 行：1=G, 6=E, 4=C, 5=D。那么 G 调的和弦就是 G - Em - C - D。
