# 第五课 大小调音阶及调号

## 音阶

- 大调音阶：全全半全全全半。
- 自然小调音阶：全半全全全半全。
- 和声小调：自然小调升第 7 级音叫和声小调。
- 旋律小调：自然小调仅仅在上行时升第 6、7 级音叫旋律小调。
- 爵士乐：爵士乐框架内旋律小调是自然小调升第 6、7 级音，下行也升。
