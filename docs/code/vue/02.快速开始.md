# 第二章 快速开始一个 Vue 应用

## 搭建 Vue 3 工程化项目

### 安装

使用 npm 创建基于 Vite 的 Vue 项目 `npm create vue@latest`。

在 VS Code 中安装「Vue - Official」和「Vue VSCode Snippets」两个插件。

### 规范

```txt
├── src
│   ├── api            数据请求
│   ├── assets         静态资源
│   ├── components     组件
│   ├── pages          页面
│   ├── router         路由配置
│   ├── store          vuex 数据
│   ├── utils          工具函数
│   └── views          页面
```
